# ENG 1005 · Grammar Cheat Sheet (One Page)
### Quick reference for the Grammar & Mechanics Exam

| Topic | Rule in one line | Example |
|---|---|---|
| **Independent clause** | Complete thought, stands alone. | *The butler vanished.* |
| **Subordinate clause** | Starts with *because/although/when/which*; can't stand alone. | *Although the alibi seemed airtight…* |
| **Fragment** | A piece missing a subject, verb, or complete thought — attach or complete it. | ✗ *Hiding in the shadows.* → ✓ *The killer was hiding in the shadows.* |
| **Comma splice** | Two sentences joined by only a comma — add a conjunction, semicolon, or period. | ✗ *The lights went out, no one screamed.* → ✓ *…out, but no one screamed.* |
| **Run-on / fused** | Two sentences with no punctuation — separate them. | ✗ *The clock struck twelve the body was found.* → ✓ *When the clock struck twelve, the body was found.* |
| **Comma: intro** | Comma after an opening phrase/clause. | *After the storm passed, they returned.* |
| **Comma: series** | Commas between items, including before *and*. | *a ticket, a watch, and a glove* |
| **Comma: extra info** | Pair of commas around non-essential info. | *Christie, who wrote the novel, …* |
| **Semicolon ( ; )** | Joins two complete, related sentences. | *The evidence was clear; the jury agreed.* |
| **Colon ( : )** | Complete sentence + list or explanation. | *One rule: never trust a calm witness.* |
| **Coordination** | Equal ideas: comma + *and/but/or/so/yet*. | *The alarm failed, so he slipped in.* |
| **Subordination** | Unequal ideas: *although/because/when/which*. | *Although it was locked, he got in.* |
| **Parallelism** | List items share the same form. | *clever, observant, and cautious* |
| **Dangling modifier** | Opening phrase must describe the next word. | ✗ *Covered in dust, the detective found it.* → ✓ *…found the diary, which was covered in dust.* |
| **Misplaced modifier** | Put the modifier next to what it describes. | ✓ *We solved almost every clue.* (not *almost solved*) |
| **Agreement** | *Each/every/everyone/neither* = singular. | *Each suspect **has** an alibi.* |
| **Collective noun** | *Jury/team* = singular. | *The jury reached **its** verdict.* |
| **Pronoun case** | After a preposition use *me/him/her/them*. | *Between you and **me**…* |

**Three-second checks**
- **Splice/run-on?** Could a period replace the comma (or the gap)? → fix it.
- **Colon?** Is there a complete sentence *before* it? → required.
- **Parallel?** Do the listed items all start with the same kind of word? → they should.
- **Pronoun?** Drop the other person: "to *me*" sounds right, "to *I*" doesn't.
